# Rust solutions to Advent of Code 2018

```
Advent Of Code 2018

USAGE:
    aoc-2018 [FLAGS] -d <day> -f <file>

FLAGS:
    -2               Solve part2
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -d <day>         which day
    -f <file>        file name
```

Example:

```
aoc-2018 -d 1 -f inputs/day1.txt -2
```
